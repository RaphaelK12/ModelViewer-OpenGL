# ModelViewer-OpenGL
OBJ model visualizer in OpenGL





# Commands

## Model Rotations (X,Y,Z)

(P,p),(Y,y),(R,r)

## Model Translations (X,Y,Z)

(L,l),(D,d),(N,n)

## Model Export

w

## Fog effect

(F,f)

## Exit

q

## Reset

s

## Camera Translations

X and Y axis: Arrows

Z axis: (I,i)

## Projection Type Switch (Ortographic:default,Perspective)

(V,v)

## Camera Rotation (X,Y,Z)

(T,t),(A,a),(C,c)
